# This Is Haringey — Project Context

## What this project is
A community directory for the London Borough of Haringey covering local
businesses, events, and neighbourhood discovery.

## Brand — quick reference
- **Logo**: "This Is Haringey." — Inter ExtraBold 800, full stop in Terracotta
- **Typeface**: Inter only — no other typefaces, ever
- **Primary accent**: Terracotta #E05A2B
- **Backgrounds**: White #FFFFFF, Cream #F7F5F0
- **Secondary accent**: Sage #6B7C6E
- **Creative concept**: "Many voices, one place"
- **Personality**: Warm, enthusiastic, community-led, modern

## Full Design Language System
For complete brand guidance — colours, typography, components, spacing,
tone of voice, and logo usage — read @docs/DLS.md

## Always apply when building UI or writing copy
- font-family: 'Inter', Arial, sans-serif — nothing else
- Tight tracking (−0.03em to −0.04em) on all headings 20px+
- Wide tracking (+0.08em to +0.12em) on uppercase labels
- 8px base spacing unit
- Terracotta sparingly — accent only, not everywhere
- Specific over generic — use real place names (N8, N17, Wood Green)
